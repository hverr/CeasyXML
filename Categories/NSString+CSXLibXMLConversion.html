NSString(CSXLibXMLConversion) Category Reference

Declared in NSString+CSXLibXMLConversion.h

Overview

This category extends the functionallity of NSString with several methods to easily convert an NSString to an xmlChar* buffer and to create strings used when working with LibXML.

Class Methods

indentationWithNewlines:level:

Creates a new string with the specified amount of new lines and the specified amount of tabs.

+ (id)indentationWithNewlines:(NSInteger)newlines level:(NSInteger)level

Parameters

newlines

The amount of newlines to append at the beginning of the string.

level

The number of tabs to append to the string.

Discussion

Creates a new string with the specified amount of new lines and the specified amount of tabs.

Note that one tab is printed out as four spaces by default. By editing the source code and recompiling the project, you can change this behaviour.

Copies the content of an NSString to an allocated buffer of xmlChar’s.

When you are done with the buffer you should use free(3) to free the allocated memory.
