empty

Specifies wether the element is an empty element or not.

@property (nonatomic, assign) BOOL empty

Discussion

Specifies wether the element is an empty element or not.

If this is set to YES, the element should look like this:

 <element />

If this is not true, the element should look like this:

<element>SOME_CONTENT</element>

unique

Specifies wether the element is unique or not.

@property (nonatomic, assign) BOOL unique

Discussion

Specifies wether the element is unique or not.

If this is set to NO, the instance variable of the parent should be an NSMutableArray, as the parser will set the instance variable to an instance of NSMutableArray containing all the elements.

If this is set to YES, the parser will just set the instance variable to an instance of the content type.
