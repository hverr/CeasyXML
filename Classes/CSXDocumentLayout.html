CSXDocumentLayout Class Reference

Inherits from NSObject
Declared in CSXDocumentLayout.h

Overview

CSXDocumentLayout represents one possible layout of a document.

The tagname for this element is layout. It can be a root element of a layout document, but it can also be a subelement of a document element used by the class CSXLayoutList.

Exceptions

  • CSXDocumentLayoutInvalidClassException

    Thrown when the class specified in the XML layout document could not be found.

initWithLayoutDocument:error:

Initiliaze a document from a layout document.

- (id)initWithLayoutDocument:(NSString *)doc error:(NSError **)err

Parameters

doc

The path to the layout document.

err

A pointer to a variable that can hold an error if one occurs. Pass NULL if you are not interested in errors. See the errors of CSXXMLParser for more information.

Return Value

An instance of CSXDocumentLayout or nil if an error occurred.

Discussion

Initiliaze a document from a layout document.

The layout document should have layout as the root element tagname. The root element can contain subelements and attributes.

 <?xml version="1.0" ?>
<layout name="root-element-name">
    <element name="myname">
    </element>
</layout>
